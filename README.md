# KeyForge
KeyForge（代号“键铸”）是一款功能强大、设计优雅的后台按键测试工具。它就像一个“键盘使者”，能够在您电脑上运行的任何窗口（尤其是那些被最小化或后台运行的游戏、软件）中模拟键盘输入，而不会干扰您当前的鼠标和键盘操作。  无论您是游戏玩家（想测试游戏连招、挂机脚本）、软件测试工程师（想自动化测试软件界面）、还是系统管理员（想批量操作多个窗口），KeyForge都能为您提供一个强大、可靠且易于使用的解决方案。它不仅支持从A到Z的所有字母和数字，还支持F1-F12功能键、方向键、组合键（如Ctrl+C），并提供了15种不同的后台按键发送方式，以应对各种复杂场景下的“按键无效”问题。  核心理念： 将最专业的后台技术，包装成最简单易用的界面，让每一位用户都能成为自己电脑的“指挥家”。
